## 1. 자기소개
안녕하세요, 데이터 기반의 문제 해결에 강점을 가진 정보통계학과 안준현입니다. 저는 R을 메인 언어로 사용하며, SAS, Python, SPSS 등 다양한 통계 프로그래밍 환경에 익숙합니다.

실제 데이터를 다루는 프로젝트 경험을 통해 실무 역량을 길러왔습니다. 예를 들어, '당뇨병 환자 분석' 프로젝트에서는 임상 데이터를 다루며 복잡한 데이터 전처리 및 정제 작업을 수행했고, 특히 관찰 데이터의 편향을 보정하기 위해 성향점수 매칭(PSM)과 같은 통계 기법을 적용하며 모델의 신뢰도를 높이는 경험을 했습니다. 또한, '미세먼지 예측' 프로젝트에서는 명확한 가설을 기반으로 실험을 설계하고, 선형회귀, 결정트리, KNN 등 다양한 모델의 성능을 비교 분석하며 최적의 방법론을 도출하는 과정을 주도했습니다.

이러한 분석 역량과 더불어, 태권도 및 축구 동아리 활동을 통해 유연한 소통 능력과 공동의 목표를 달성하는 협업 능력을 체득했습니다. 저의 기술적 역량과 소프트 스킬을 바탕으로 조직에 기여하는 데이터 분석가로 성장하고 싶습니다.

## 2. 학습 목표
저의 현재 R 기반 통계 모델링 및 데이터 전처리 역량을 공고히 하는 동시에, Python을 활용한 고급 머신러닝 및 딥러닝 알고리즘으로 전문성을 확장하는 것이 목표입니다.

단순히 모델을 적용하는 것을 넘어, 임상, 환경, 비즈니스 등 더 복잡하고 다양한 문제를 통계적 관점에서 해결할 수 있는 데이터 분석 전문가로 성장하기 위해 꾸준히 학습하고 있습니다.

## 4. 프로젝트 경험

### 당뇨병 환자 로지스틱 회귀모델 분석
* **역할:** 데이터 분석, 발표 및 파워포인트 제작
* **주요 역량:**
    * R을 활용한 실제 임상 데이터 전처리 및 정제
    * 관찰 데이터의 선택 편향(Selection Bias) 보정을 위한 **성향점수 매칭(PSM)** 적용
    * 로지스틱 회귀모델을 통한 통계적 유의성 분석

### 미세먼지 예측 회귀모델
* **역할:** 데이터 분석, 발표 및 파워포인트 제작
* **주요 역량:**
    * R을 활용한 기상 및 지리 데이터 통합 및 가공
    * '기상 vs 기상+지리' 변수군에 따른 **가설 기반의 실험 설계**
    * **다양한 회귀 모델(선형회귀, 결정트리, KNN) 성능 비교** 및 최적 모델 선정
